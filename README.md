# Projeto Empresa de Ações

## Descrição

Projeto de uma empresa de Ações - Compra e Venda

**Características do Projeto:**
- Linguagem: JAVA
- Backend: Java Spring MVC/Rest
- Banco de Dados: MongoDB
- Fila de Comunicação Assíncrona: RabbitMQ
